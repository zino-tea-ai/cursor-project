# MyFitnessPal 产品截图分析

> 产品经理专用 | 200张截图 | 按流程命名 | 可直接导入Figma

## 📁 项目结构

```
MFP_Analysis/
├── 📁 Screens/                 # ⭐ 所有截图（200张，扁平结构）
├── 📁 MFP_Screens_Downloaded/  # 原始下载备份
├── 📄 download_screens.py      # 下载脚本
├── 📄 classify_screens.py      # 分类脚本
├── 📄 rename_screens.py        # 重命名脚本
├── 📄 requirements.txt         # Python依赖
└── 📄 README.md
```

## 🎯 快速使用

**导入 Figma**: 直接打开 `Screens` 文件夹，全选拖入 Figma 即可
